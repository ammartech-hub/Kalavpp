# Git Push Instructions & File Manifest

## Instructions to Push to GitHub
Run these commands in your VS Code terminal (Ctrl+`):

1. Initialize git (if not already done):
   git init

2. Add all files:
   git add .

3. Commit changes:
   git commit -m "Initial commit including file manifest"

4. Add remote (replace YOUR_REPO_URL):
   git remote add origin https://github.com/YOUR_USERNAME/aphelion-event.git

5. Push:
   git branch -M master
   git push -u origin master
